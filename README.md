##  Sobre o Projeto

O objetivo deste projeto é validar, por meio de testes **End-to-End (E2E)**, o comportamento das principais funcionalidades da aplicação:

 https://automationpratice.com.br/

Os testes foram desenvolvidos com foco em:

* Clareza e organização
* Cobertura de fluxos reais
* Validação de sucesso e comportamento esperado
* Geração de evidências (screenshots e relatórios)

---

##  Cenários Automatizados

###  Login

* Validação de login com credenciais válidas
* Verificação de mensagem de sucesso
* Redirecionamento após autenticação

---

###  Cadastro

* Preenchimento do formulário de registro
* Criação de novo usuário
* Validação de sucesso no cadastro
* Redirecionamento para área logada

---

###  Checkout

* Login no sistema
* Busca de produto
* Adição ao carrinho
* Preenchimento dos dados
* Finalização da compra
* Validação de confirmação
